Программа предназначена для сортировки файлов на флешке FAT32 с целью установки корректного порядка воспроизведения музыкальных треков стандартным проигрывателем ГУ автомобилей HAVAL (как минимум H9 Gen2). Флешка обязана быть отформатирована в FAT32. Программа сортирует файлы в таблице директорий файловой системы в соответствии с их именами и устанавливает в каждом подкаталоге каждому следующему файлу время (создания, модификации и доступа) на 1 минуту меньше предыдущего.

Программа не требует установки: копируем исполняемый файл в любое место на диске, выполняем в консоли haval_sort.exe <путь_к_флешке> (например ">haval_sort.exe E:\\").
При запуске без параметров выведет информацию о программе и использовании.

В подкаталоге bin находится собранный исполняемый файл. Сборка производилась при помощи mingw на Windows 10. При желании можно собрать бинарник из исходника самостоятельно.
Программа предназначена для сборки и выполнения только в среде windows! На других ОС работать не будет!

После выполнения сортировки крайне рекомендуется изменить метку флешки (например при помощи команды label), есть подозрение, что ГУ кеширует списки файлов и не перечитывает, из за этого результаты сортировки могут не отображаться.
